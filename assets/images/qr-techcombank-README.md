# Hướng dẫn tạo QR Code Techcombank cho Bravemath

## Bước 1: Tạo QR Code từ app Techcombank
1. Mở app Techcombank trên điện thoại
2. Vào phần "Nhận tiền" hoặc "QR Pay"
3. Tạo mã QR với thông tin tài khoản của bạn
4. Xuất ảnh QR code

## Bước 2: Lưu file
- Lưu file QR code với tên: `qr-techcombank.png`
- Đặt vào thư mục: `assets/images/qr-techcombank.png`

## Thông tin tài khoản mẫu (thay bằng thông tin thật):
- Ngân hàng: Techcombank
- Chủ tài khoản: Bùi Quang Chiến
- Số tài khoản: [Thay bằng số thật]

## Lưu ý:
- QR code nên có độ phân giải cao (ít nhất 300x300px)
- Nền trắng, mã QR đen rõ ràng
- Test quét được trước khi upload
